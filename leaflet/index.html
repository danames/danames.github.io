Web Mapping Service Examples

These examples use WMS services hosted on HydroShare. See HydroShare documentation for more information on creating resources. If you load a shapefile or GeoTIFF on HydroShare it will automatically create WMS end points for you in geoserver.hydroshare.org.

  • Basemap plus a WMS point layer from HydroShare GeoServer with a simple point styling using an SLD file - rotated square
  • Basemap plus a WMS point layer from HydroShare GeoServer with a simple point styling using an SLD file - transparent triangle
