Simple Leaflet Map Demonstration

By Dr. Dan Ames - May 22, 2024

Sample Code

            
<!DOCTYPE html>
<html>
<head>
    <title>Simple Leaflet Map</title>
    <!-- Leaflet CSS -->
    <link rel="stylesheet" href="https://unpkg.com/leaflet@1.9.4/dist/leaflet.css" integrity="sha256-p4NxAoJBhIIN+hmNHrzRCf9tD/miZyoHS5obTRR9BMY=" crossorigin="" />
    <style>
        #map { height: 400px; }
    </style>
</head>
<body>

    <div id="map"></div>

    <!-- Leaflet JavaScript -->
    <script src="https://unpkg.com/leaflet@1.9.4/dist/leaflet.js" integrity="sha256-20nQCchB9co0qIjJZRGuk2/Z9VM+kNiyxNV1lvTlZBo=" crossorigin=""></script>
    <script>
        var map = L.map('map').setView([40.7128, -74.0060], 13); // New York City

        L.tileLayer('https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png', {
            attribution: '© <a href="https://www.openstreetmap.org/copyright">OpenStreetMap</a> contributors'
        }).addTo(map);
    </script>
</body>
</html>

Key Points to Remember

  • Include Leaflet CSS and JavaScript: You must include links to the Leaflet CSS and JavaScript files in your HTML's <head> section. These are hosted on a CDN (Content Delivery Network) for easy access.
  • Create a Map Container: You need a <div> element with a unique ID to hold your map. Set its height using CSS, or it won't be visible. Width can also be set, otherwise it defaults to 100% of its containing element.
  • Initialize the Map: Use L.map('map-id') to initialize the map, replacing 'map-id' with the ID of your map container <div>.
  • Set Initial View: Use .setView([latitude, longitude], zoomLevel) to set the initial geographical center and zoom level of the map.
  • Add a Tile Layer: Use L.tileLayer() to add a base map (tiles) to your map. OpenStreetMap is a common choice and is free to use. Remember to include the attribution.
  • Add the Tile Layer to the Map Call the .addTo(map) method on the tile layer object to add it to your map.
